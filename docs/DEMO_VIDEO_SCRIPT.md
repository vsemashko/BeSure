## Production Requirements

### Visual Assets Needed

**App Screens to Record:**
1. Feed screen (For You, Urgent, Popular tabs)
2. Question detail screen with voting
3. Create question flow (all steps)
4. Results screen with charts
5. Profile screen with stats
6. Settings/privacy screens
7. Notification center
8. Leaderboard
9. Achievements page
10. Friends/social screens

**Custom Graphics:**
- BeSure logo animations
- Point earning animations (+2, +10, etc.)
- Achievement unlock animations
- Streak fire emoji animations
- Progress bars and charts
- App Store badges
- Social media icons

**Stock Footage (if needed):**
- People looking stressed/indecisive
- People looking happy/confident
- Diverse users with phones
- Various decision scenarios (shopping, choosing outfits, etc.)

### Technical Specifications

**Resolution:**
- 4K minimum (3840 x 2160) for future-proofing
- Export in 1080p for most platforms

**Frame Rate:**
- 30 fps (standard)
- Or 60 fps for smoother app screen recordings

**Aspect Ratios:**
- 16:9 (Horizontal) - YouTube, website
- 9:16 (Vertical) - Instagram, TikTok, Stories
- 1:1 (Square) - Facebook, Instagram feed
- 4:5 (Vertical) - Instagram feed optimized

**Audio:**
- Professional voiceover (clear, friendly, energetic)
- Background music (licensed or royalty-free)
- Sound effects for app interactions (subtle)

### Voiceover Guidelines

**Tone:**
- Friendly and approachable
- Confident but not pushy
- Energetic but not overwhelming
- Conversational, not corporate

**Pace:**
- Moderate to slightly fast
- Allow time for visuals to breathe
- Emphasize key phrases
- Natural pauses at transitions

**Style:**
- Gender neutral option preferred
- Clear pronunciation
- Slight enthusiasm without being over-the-top

## Screen Recording Guide

### App Screen Recording Setup

**Device:**
- iPhone 14/15 Pro or newer (for iOS version)
- Google Pixel or Samsung Galaxy (for Android version)
- Use screen recording feature built into devices

**Settings:**
- Enable "Do Not Disturb" to avoid notifications
- Full brightness for clean recording
- Hide status bar if possible
- Use demo account with curated content

**Demo Content Preparation:**
- Create sample questions with high-quality images
- Prepare diverse question types (fashion, tech, travel, food, etc.)
- Stage realistic vote counts and percentages
- Set up user profiles with appropriate stats
- Have achievements/badges ready to unlock

### Recording Tips

1. **Clean recordings:** No shaky movements, smooth transitions
2. **Timing:** Hold on each screen for 2-3 seconds minimum
3. **Interactions:** Show natural tap/swipe gestures
4. **Highlights:** Use subtle animations or arrows to guide viewer attention
5. **Consistency:** Use same demo account throughout

## Editing Guidelines

### Visual Style
- **Clean and modern:** Match app's design aesthetic
- **Smooth transitions:** Fade, slide, or app-style transitions
- **Minimal effects:** Don't overdo it
- **On-screen text:** Bold, sans-serif, easy to read
- **Color scheme:** Match BeSure brand colors

### Pacing
- **Short version:** Quick cuts, dynamic pacing
- **Long version:** Allow moments to breathe, clear explanations
- **Never rush:** Viewers should understand each section

### Text Overlay Best Practices
- **Readable:** High contrast, large enough font
- **Brief:** 3-7 words maximum per text overlay
- **Timed well:** Appears when mentioned, stays long enough to read
- **Positioned well:** Doesn't cover important UI elements

## Call-to-Action Optimization

### End Screen (Last 5-10 seconds)

**Must Include:**
- BeSure logo
- "Download Now" or similar CTA
- App Store badges (iOS and Android)
- Star rating (if available)
- Optional: QR code for instant download

**Best Practices:**
- Hold end screen for full 5-10 seconds
- Keep CTA clear and prominent
- Add subtle animation to draw eye
- Include website URL if applicable

## Distribution Versions

### Platform-Specific Edits

**YouTube:**
- Full 2-3 minute version
- Include end screens with links
- Optimize title and description for SEO
- Add chapters/timestamps

**Instagram/Facebook:**
- 60-second version
- Vertical (9:16) or square (1:1)
- Captions/subtitles (many watch without sound)
- First 3 seconds must hook viewer

**TikTok:**
- 30-60 second version
- Must start with strong hook
- Trendy music option
- Native feel (less polished ok)

**App Store/Google Play:**
- 30-second version
- Focus on key features
- Must follow store guidelines
- Show actual app interface

**Website Hero:**
- 30-60 second loop
- Can be without voiceover
- Background music ok
- Emphasize visual appeal

## Budget Estimate

### DIY / Low Budget ($0 - $500)
- **Voiceover:** Use Fiverr ($50-100)
- **Music:** Free library or Uppbeat ($10/month)
- **Screen recording:** Use your own device (free)
- **Editing:** Use free software (iMovie, DaVinci Resolve)
- **Stock footage:** Free sources (Pexels, Unsplash)
- **Total:** $50-200

### Professional / Medium Budget ($500 - $2,000)
- **Voiceover:** Professional voice actor ($200-400)
- **Music:** Epidemic Sound or Artlist ($15-25/month)
- **Videographer:** Hire for custom shots ($300-600)
- **Editor:** Professional editing ($400-800)
- **Motion graphics:** Custom animations ($200-400)
- **Total:** $1,100-2,200

### Premium / High Budget ($2,000 - $10,000)
- **Production company:** Full service
- **Professional actors:** Real user testimonials
- **Studio shooting:** High-end production
- **Custom animations:** 3D and advanced effects
- **Multiple versions:** All platform variations
- **Total:** $2,000-10,000+
